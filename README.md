# Java Restaurant Menu System

A simple desktop restaurant menu application built using Java Swing.

This project simulates a basic restaurant ordering system where users can view menu items, select food, and generate a bill.

## Features
- View restaurant menu items
- Select food items
- Add items to order
- Generate total bill
- Simple graphical user interface

## Technologies Used
- Java
- Java Swing
- NetBeans IDE

## Project Structure
- Menu classes for displaying food items
- Order handling for selected items
- GUI built using Swing components (JFrame, JButton, JLabel)
